# 루틴 앱 개발 프로젝트

**루틴 앱**은 먼 미래 계획보다는 당장 중요한 일 하나에 일주일 동안 집중하는 아이디어에서 시작된 프로젝트입니다.

## 주요 구현 기능 및 기술 스택

### 앱 심사 및 배포
현재 앱 스토어 심사 과정을 진행 중이며, 여러 번의 실패를 겪었으나 꾸준히 도전하고 있습니다.

### 애니메이션 효과 및 UI 향상
앱의 재미와 사용자 경험을 높이기 위해 다양한 스택과 애니메이션 효과를 사용하였습니다.

### Provider를 사용한 상태 관리
- Provider 패턴을 활용하여 전역 데이터 구조를 설계하고, 이를 기반으로 앱 전반에 걸쳐 데이터를 관리하였습니다.
- 효율적인 상태 관리를 구현하였습니다.

### 사용자 데이터 입력 Form 구현
- 사용자로부터 데이터를 입력받기 위해 다양한 Form을 구현하였습니다.
- 유효성 검사 및 입력 필드의 사용자 친화적인 디자인을 고려하였습니다.

### UI 디자인 (Figma)
- Figma를 사용하여 앱의 UI를 디자인하였습니다.
- 직관적이고 사용하기 쉬운 인터페이스를 목표로 하였습니다.

### 사용성 향상 (GestureDetector와 ElevatedButton)
- GestureDetector와 ElevatedButton을 적절하게 사용하여 사용자 경험을 개선하였습니다.
- 상호작용을 보다 직관적으로 만들었습니다.

### Navigation을 통한 화면 전환 및 데이터 전달
- Flutter의 Navigation 기능을 사용하여 화면 간 이동 및 데이터 전달을 구현하였습니다.
- 자연스러운 사용자 흐름을 유지하였습니다.

### Snackbar를 사용한 알림 기능
- Snackbar를 사용하여 사용자에게 간단한 알림을 제공하는 기능을 구현하였습니다.
- 중요한 정보나 피드백을 즉각적으로 전달할 수 있도록 하였습니다.

### 플러그인 및 외부 패키지 통합
- pub.dev와 같은 패키지 저장소를 사용하여 프로젝트에 필요한 외부 패키지를 검색하고 통합하는 경험이 있습니다.

## 사용한 도구 및 기술

- **백엔드**: Firebase
- **상태 관리**: Provider
- **UI 디자인**: Figma
- **프로그래밍 언어**: Dart (Flutter)
- **기타**: GestureDetector, ElevatedButton, Navigation, Snackbar
